# lmrl - 圣经搜索和音频工具

## 项目描述

`lmrl` 项目是一个带有 Web 前端和 Go 后端的圣经搜索工具。它允许用户搜索圣经经文或关键词并查看结果。它还包括一个搜索历史记录功能。 "lmrl" 这个名字代表 "灵命日粮"，反映了其提供每日灵性喂养的初衷。

## 特点

- 搜索圣经经文和关键词。
- 在用户友好的界面中查看搜索结果。
- 维护搜索历史记录以便轻松回顾。
- 收听音频内容。

## 如何运行项目（开发环境）

要在本地设置和运行项目进行开发，请导航到项目的根目录并执行以下命令：

```bash
make dev
```

此命令将：

1.  进入 `frontend` 目录。
2.  使用 `npm install` 安装前端依赖。
3.  使用 `vite build` 构建前端应用程序。
4.  将必要的前端资产复制到 `router/frontend/assets` 目录。
5.  构建 Go 后端应用程序。
6.  启动后端服务器，该服务器将提供前端服务并处理 API 请求。

应用程序通常可在 `http://localhost:3001/lmrl/search` 访问。

## 如何运行项目（生产/部署）

对于生产部署，您可以使用 `make build` 和 `make deploy` 命令。

### 构建

编译后端应用程序以用于生产环境：

```bash
make build
```

### 部署

部署应用程序（这通常涉及将编译后的二进制文件和资产复制到服务器，通常与 Nginx 等反向代理结合使用）：

```bash
make deploy
```

_（注意：具体的部署配置，例如 Nginx 设置，超出了本通用 README 的范围，应根据您的环境单独配置。）_

## 最近的 UI 改进

最近的工作重点是提高用户界面的稳定性，特别解决了在与搜索历史记录交互时，左侧（搜索历史记录）和右侧（搜索结果）面板会“跳动”或意外改变大小的问题。这些改进确保了更稳定和可预测的布局。
